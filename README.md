# HanShan Blog

## 简介

本仓库为Github Pages仓库，利用Github Actions和Hexo静态模板生成最终的博客网站

地址为[HanShan Blog (hanshansite.com)](https://www.hanshansite.com/)

## 搭建技术

* Github Pages
* Github Actions
* Hexo 6.3.0
* Hexo Theme Redefine
